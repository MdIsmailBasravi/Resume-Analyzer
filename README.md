# 📄 Resume Analyzer

A web application that analyzes resumes and matches them with job descriptions.

## 🚀 Features

* Upload PDF resume
* Extract resume text
* Resume scoring system
* Skill detection
* Job matching
* Suggestions for improvement

## 🛠️ Tech Stack

* Python
* Streamlit
* PyPDF2

## ▶️ How to Run

pip install -r requirements.txt
streamlit run app.py

## 📸 Output

* Resume Score
* Skills Found / Missing
* Job Match %
* Suggestions
